
# Example Site

This example is the documentation site for Hastie.

The source files are in: [pages/](https://github.com/mkaz/hastie/tree/trunk/example/pages) directory.

The layout files are up a level in: [themes/docs](https://github.com/mkaz/hastie/tree/trunk/themes/docs) directory.

The directories are specified in the [hastie.conf](https://github.com/mkaz/hastie/tree/trunk/example/hastie.conf) file.

To generate the site, `cd` to this directory and run `hastie` binary. The generated files will be put in `output/`
